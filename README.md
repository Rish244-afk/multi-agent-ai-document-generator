# 🤖 Multi-Agent AI Document Generator

## 🚀 Overview
A multi-agent AI system where multiple AI agents collaborate to generate high-quality documents using Google Gemini API.

## 🧠 Architecture
User Input → Writer Agent → Reviewer Agent → Improver Agent → Final Output

## ✨ Features
- Multi-agent collaboration (Writer, Reviewer, Improver)
- Prompt chaining for improved output
- Metrics (word count & improvement score)
- Streamlit UI
- Downloadable results

## ⚙️ Tech Stack
- Python
- Streamlit
- Google Gemini API

## ▶️ Run Locally
pip install -r requirements.txt  
python -m streamlit run app.py
